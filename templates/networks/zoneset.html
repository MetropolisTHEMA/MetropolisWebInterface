Attributes

  • project Project: Project the ZoneSet instance belongs to.

  • srid str: Spatial Reference System Identifier used for the coordinates of the zones.
  • name str: Name of the instance.
  • comment str: Description of the instance (default is '').
  • ags set of str: Tags describing the instance, used to search and filter the instances.
  • date_created datetime.date: Creation date of the ZoneSet.
